******************************
*Java Stepping Stone problem
******************************

The stepping stone problem was solved in Java using graphs (OOP paradigm).

findOptimalTransport.java is the entry point of the program, execute it to run the prgoram. 
To run properly findOptimalTransport.java needs a file of a similar format to that found in the prob.txt.
probSolution.txt represents an output example of running the program with prob.txt
