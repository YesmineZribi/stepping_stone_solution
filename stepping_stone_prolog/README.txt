*****************************
*Stepping Stone in Prolog
*****************************

Stepping stone solution implementation in Prolog (logic programming)

minimumTransportCost(C) in order to run the program. 
You will be prompted to enter the name of two files: 
(1) The file describing the problem
(2) The file describing the initial solution
Wait for the matrix solution to appear as well as the Marginal total cost in the end 
(Note: the 10x10 matrix works but it takes a few seconds to display the results)
